Cargar datos

En esta parte se cargan tanto los datos de expresión como las imágenes para así crear los objetos de la clase ‘SpatialExperiment’ que permite aplicar el pipeline de Bioconductor a nuestras muestras de Spatial transcriptomics.

#Función para cargar una muestra
load_spe <- function(sample, sample_id, dir) {
  
  # Conteos
  fnm <- file.path(dir, sample, "filtered_feature_bc_matrix")
  sce <- DropletUtils::read10xCounts(fnm, sample.names = sample_id, col.names = TRUE)
  
  # Filtrado de barcodes
  filternm <- file.path(dir, sample, "filtered_feature_bc_matrix", "filtered_barcodes.csv")
  filtered_barcodes <- read_csv(filternm, show_col_types = FALSE)
  sce_sub <- sce[, colnames(sce) %in% filtered_barcodes$Barcode]
  
  # Imagen
  img <- readImgData(
    path = file.path(dir, sample, "spatial"),
    sample_id = sample_id
  )
  
  # Coordenadas espaciales
  fnm <- file.path(dir, sample, "spatial", "tissue_positions.csv")
  xyz <- read.csv(fnm)
  xyz <- xyz[match(colnames(sce_sub), xyz$barcode), ]
  
  # Metadata de genes
  rd <- S4Vectors::DataFrame(symbol = rowData(sce)$Symbol)
  
  # Objeto SpatialExperiment
  spe <- SpatialExperiment(
    assays = list(counts = assay(sce_sub)),
    rowData = rd,
    colData = DataFrame(xyz),
    spatialCoordsNames = c("pxl_col_in_fullres", "pxl_row_in_fullres"),
    imgData = img,
    sample_id = sample_id
  )
  
  return(spe)
}

Control calidad

Aplicamos el control de calidad, que permitirá eliminar spots de baja calidad, en base a una serie de características: - Tamaño de la librería - Número de genes expresados - Proporción de conteos de genes mitocondriales, ya que un número elevado es indicador de daño celular

Se calculan las siguientes métricas:

  • sum: suma total de conteos en cada spot

  • detected: número de genes con un número de conteos > 0

  • subsets_mito_sum: número de conteos de genes mitocondriales

  • subsets_mito_detected: número de genes mitocondriales con un número de conteos > 0.

  • subsets_mito_percent: porcentaje de expresión de los genes mitocondriales

  • total: lo mismo que sum (número total de conteos por spot)

Umbrales por MAD

En este caso se aplica la función isOutlier, del paquete scuttle para obtener un umbral que se basa en la desviación absoluta mediana (MAD). Consiste en calcular la distancia de cada valor a la mediana de cada métrica, en este caso conteos totales, genes detectados, y porcentaje de genes mitocondriales. De todas esas distancias se obtiene la mediana (MAD), y el umbral será, en nuestro caso, quitar los puntos que se encuentren a una distancia mayor de 3 veces la MAD de la mediana.

low.lib <- isOutlier(

    log = TRUE,

low.detected <- isOutlier(

Normalización

A pesar de que están todas las muestras en el mismo objeto, se aplica la normalización por muestra. Y es que como la normalización tiene como objetivo eliminar las diferencias técnicas (profundidad de secuenciación o eficiencia de capturas) y las muestras se han secuenciado en slides distintos, van a tener sesgos técnicos distintos, y, por tanto, se deben normalizar por separado.

El primer paso de la normalización es calcular el size factor de cada spot. Este se calula haciendo el tamaño de la librería de cada spot (número total de cuentas) entre la media total. El conteo de cada gen de cada spot se divide entre este número. Por último, se aplica log2(x+1).

run_find_markers <- function(spe, top_n = 2, min_spots = 5) {
  
  # Identifica clusters con suficientes spots
  cluster_counts <- table(spe$BayesSpace
                          )
  clusters_ok    <- names(cluster_counts[cluster_counts >= min_spots])
  
  n_excluidos <- sum(cluster_counts < min_spots)
  if (n_excluidos > 0) {
    message("[ MARKERS ] Excluyendo ", n_excluidos, " cluster(s) con menos de ",
            min_spots, " spots: ",
            paste(names(cluster_counts[cluster_counts < min_spots]), collapse = ", "))
  }
  
  # Filtra el objeto para quedarte solo con los clusters válidos
  spe_sub <- spe[, spe$BayesSpace %in% clusters_ok]
  spe_sub$BayesSpace <- droplevels(spe_sub$BayesSpace)  # elimina niveles vacíos
  
  mgs <- findMarkers(
    spe_sub,
    test.type = "wilcox",
    groups    = spe_sub$BayesSpace,
    direction = "up"
  )
  
  top <- lapply(mgs, function(df) rownames(df)[df$Top <= top_n])
  top <- unique(unlist(top))
  message("[ MARKERS ] Genes marcadores seleccionados: ", length(top))
  
  pbs <- aggregateAcrossCells(
    spe_sub,
    ids               = spe_sub$BayesSpace,
    subset.row        = top,
    use.assay.type    = "logcounts",
    statistics        = "mean"
  )
  
  mtx <- t(assay(pbs))
  colnames(mtx) <- rowData(pbs)$symbol
  
  
  # Tamaño de fuente adaptado al número de genes
  n_genes <- length(top)
  fontsize_col <- dplyr::case_when(
    n_genes <= 20  ~ 12,
    n_genes <= 30  ~ 11,
    n_genes <= 40  ~ 10,
    n_genes <= 50  ~ 9,
    TRUE           ~ 4
  )
  
  hm <- pheatmap(
    mat          = mtx,
    scale        = "column",
    fontsize_col = fontsize_col,   # genes (columnas)
    fontsize_row = 10              # clusters (filas), estos son pocos, tamaño fijo
  )
  print(hm)
    
  return(invisible(list(mgs = mgs, heatmap = hm, top_ids = top)))
}
